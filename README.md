# ecobiciweather-databricks-etl-dev
ecobiciweather-databricks-etl-dev is a Databricks ETL project built with Medallion architecture, integrating EcoBici and weather data in Azure. It uses PySpark, Unity Catalog, Managed Identity, and GitHub Actions to ingest, transform, secure, and publish curated datasets for analytics and dashboarding on bike usage vs weather conditions.
